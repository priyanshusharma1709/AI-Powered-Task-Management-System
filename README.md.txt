# AI-Powered Task Management System

This project uses Machine Learning and Natural Language Processing (NLP)
to classify and prioritize tasks automatically.

## Technologies Used
- Python
- NLP (NLTK)
- TF-IDF Vectorization
- Machine Learning (Naive Bayes, SVM)
- Jupyter Notebook

## Purpose
This project was developed for academic learning and understanding
of AI-based task classification systems.
